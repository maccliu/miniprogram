# 全局样式

`app.wxss`文件存放全局样式。

## 宽度

```css
.w-1  {width: 8.333333%;}
.w-2  {width: 16.666666%;}
.w-3  {width: 25%;}
.w-4  {width: 33.333333%;}
.w-5  {width: 41.666666%;}
.w-6  {width: 50%;}
.w-7  {width: 58.333333%;}
.w-8  {width: 66.666666%;}
.w-9  {width: 75%;}
.w-10 {width: 83.333333%;}
.w-11 {width: 91.666666%;}
.w-12 {width: 100%;}
```

### 5栏宽度系列

```css
.w-5-1 {width: 20%;}
.w-5-2 {width: 40%;}
.w-5-3 {width: 60%;}
.w-5-4 {width: 80%;}
.w-5-5 {width: 100%;}
```
